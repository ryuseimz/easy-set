# Install Links

講座で使う公式インストールリンクと画面イメージです。

## Claude Code / Claude Desktop

- Claude ダウンロード: https://claude.com/download
- Claude Desktop インストール手順: https://support.claude.com/en/articles/10065433-install-claude-desktop
- Claude Code docs: https://code.claude.com/docs

注意:

- 検索広告や非公式ミラーではなく、公式リンクからダウンロードしてください。
- Claude Desktop は macOS / Windows の OS 別ダウンロードを選びます。
- Claude Desktop の `Settings > Extensions` から desktop extensions を確認できます。
- Claude Code は desktop app からも扱えます。CLI は上級者向けのおまけです。

## Codex / ChatGPT Desktop

- Codex 公式ページ: https://openai.com/codex/
- Codex getting started: https://openai.com/academy/codex-how-to-start/
- Codex developer docs: https://developers.openai.com/codex
- ChatGPT desktop app: https://chatgpt.com/features/desktop/

注意:

- Codex は ChatGPT アカウントでサインインします。
- 講座本編では Codex デスクトップアプリを使います。
- CLI、cmux、Warp は最後の上級者向けトピックです。
