# Review Prompt

実装後に Codex / Claude Code に投げるレビュー用プロンプトです。

```text
この変更をレビューしてください。

観点:
- MVP の目的に対して余計な機能が増えていないか
- バグや壊れやすい実装がないか
- スマホ表示で崩れないか
- README の起動手順が正しいか
- secrets や API key が混入していないか
- Vercel deploy で問題になりそうな点がないか

問題があれば、重要度順に指摘してください。
修正できるものは修正してください。
修正後に lint / typecheck / test / build のうち、このプロジェクトに存在するものを実行してください。
```
