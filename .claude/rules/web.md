# Web Rules

- Prefer the framework and component patterns already present in the app.
- Verify UI in browser with screenshots for visual changes.
- Cover loading, empty, error, disabled, hover/focus, mobile, and desktop states.
- Run lint/typecheck/tests relevant to the changed files.
- Use Context7 or official docs for fast-moving APIs such as Next.js, AI SDK, Supabase, Stripe, and Vercel.
